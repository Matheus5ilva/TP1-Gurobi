  gurobi.GRBColumn

public class GRBColumn
extends java.lang.Object

Gurobi column object. A column consists of a list of coefficient, constraint pairs. Columns are used to represent the set of constraints in which a variable participates, and the associated coefficients. They are temporary objects that typically have short lifespans.

You generally build columns by starting with an empty column (using the @link GRBColumn#GRBColumn GRBColumn constructor), and then adding terms. Terms can be added individually, using @link GRBColumn#addTerm addTerm, or in groups, using @link GRBColumn#addTerms addTerms. Terms can also be removed from a column, using @link GRBColumn#remove remove.

Individual terms in a column can be queried using the @link GRBColumn#getConstr getConstr, and @link GRBColumn#getCoeff getCoeff methods. You can query the number of terms in the column using the
